# predict.py

CLI-инференс — подаёшь фото, папку или видео, модель рисует маски найденных хвостов и сохраняет результат.

## Использование

```bash
python predict.py --source photo.jpg              # одно изображение
python predict.py --source test_photos/            # папка
python predict.py --source video.mp4               # видео
python predict.py --source photo.jpg --conf 0.3    # с пониженным порогом
python predict.py --source photo.jpg --weights path/to/best.pt  # свои веса
```

## Аргументы командной строки

| Аргумент | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| `--source` | str | обязательный | Путь к изображению, папке или видео |
| `--conf` | float | 0.5 | Порог уверенности (ниже = больше детекций, но больше ложных) |
| `--weights` | str | `weights/best.pt` | Путь к файлу весов модели |

По умолчанию веса ищутся в `weights/best.pt` относительно корня проекта. Можно переопределить через `--weights`.

## Функции

### `main()`

1. **Парсинг аргументов** — `argparse` обрабатывает `--source`, `--conf`, `--weights`
2. **Загрузка модели** — `YOLO(args.weights)`
3. **Предсказание** — `model.predict()` с параметрами:
   - `save=True` — сохранять результат на диск
   - `show_labels=True` — показывать название класса
   - `show_conf=True` — показывать процент уверенности
   - `line_width=2` — толщина контура маски
   - `exist_ok=True` — перезаписывать папку результатов
4. **Вывод** — количество обработанных изображений и путь к результатам

Результаты сохраняются в `runs/predict/result/`.
