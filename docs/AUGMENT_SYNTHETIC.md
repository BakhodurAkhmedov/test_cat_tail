# augment_synthetic.py

Генерация синтетических тренировочных данных. Вырезает хвосты из реальных размеченных фотографий, аугментирует их и вставляет на случайные фоны. Результат — новые изображения с готовой YOLO-разметкой.

## Константы

- `NUM_SYNTHETIC = 50` — количество синтетических изображений для генерации
- `SEED = 42` — фиксированный seed для воспроизводимости результатов
- `IMG_SIZE = 640` — размер итогового изображения в пикселях

## Пайплайн аугментации (`tail_transform`)

Набор трансформаций из библиотеки Albumentations, применяемых к вырезанному хвосту перед вставкой:

| Трансформация | Параметры | Вероятность |
|---------------|-----------|-------------|
| Горизонтальное отражение | — | 50% |
| Поворот | ±30° | 70% |
| Масштабирование | 70%–130% | 50% |
| Яркость/контраст | ±30% | 50% |
| Цвет (HSV) | оттенок ±10, насыщенность ±30, яркость ±30 | 50% |
| Гауссово размытие | ядро 3–5 | 30% |

Трансформации комбинируются случайно — каждый раз получается уникальный вариант хвоста.

## Функции

### `load_tail_cutout(img_path, txt_path)`

Извлекает область хвоста из изображения по YOLO-разметке.

1. Читает изображение и файл разметки
2. Переводит нормализованные координаты полигона в пиксельные
3. Создаёт бинарную маску (белое = хвост, чёрное = фон)
4. Вырезает прямоугольную область вокруг хвоста с отступом 10px

**Возвращает:** вырезанный фрагмент изображения, маску хвоста, координаты полигона в системе кропа.

### `get_backgrounds()`

Собирает набор фоновых изображений из двух источников:

1. **Реальные фоны** — из папки `data/backgrounds/` (если существует)
2. **Сгенерированные** — 10 случайных однотонных изображений 640×640 с гауссовым шумом для текстуры

**Возвращает:** список фоновых изображений.

### `paste_tail_on_background(tail_crop, mask_crop, background)`

Накладывает аугментированный хвост на фон и генерирует YOLO-разметку.

1. Ресайзит фон до 640×640
2. Применяет `tail_transform` к хвосту и маске одновременно (маска трансформируется синхронно)
3. Если хвост больше фона — уменьшает его
4. Выбирает случайную позицию на фоне
5. Вставляет хвост через альфа-блендинг (плавное наложение по маске)
6. Извлекает контур маски, упрощает его и переводит в нормализованные YOLO-координаты

**Возвращает:** готовое изображение и строку YOLO-разметки.

### `main()`

Основной цикл генерации:

1. Фиксирует seed для воспроизводимости
2. Собирает все пары (изображение + разметка) из `data/images/train/`
3. Загружает фоны
4. В цикле (до 250 попыток): случайно выбирает пару и фон, вырезает хвост, вставляет на фон, сохраняет как `synthetic_0000.jpg` ... `synthetic_0049.jpg` в тренировочную папку

Защита от зацикливания: `max_attempts = NUM_SYNTHETIC * 5`.
