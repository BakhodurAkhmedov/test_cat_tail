# app/app.py

Веб-приложение на Gradio для демонстрации модели. Открывается в браузере, позволяет загрузить фото или видео кота и увидеть сегментированный хвост.

## Запуск

```bash
# по умолчанию — weights/best.pt, порт 7860
python app/app.py

# свои веса
python app/app.py --weights path/to/best.pt

# свой порт
python app/app.py --port 8080
```
Открывается на http://localhost:7860 (или на указанном порту).

## Аргументы командной строки

| Аргумент | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| `--weights` | str | `weights/best.pt` | Путь к файлу весов модели |
| `--port` | int | 7860 | Порт веб-сервера |

## Загрузка модели

Модель загружается **один раз** при старте приложения и переиспользуется для всех запросов.

## Функции

### `segment_image(image, confidence=0.3) -> np.ndarray`

Обработка одного изображения. Вызывается при нажатии кнопки "Segment" во вкладке Image.

1. Прогоняет модель на изображении с заданным порогом уверенности
2. Рисует маски и bounding box'ы через `results[0].plot()`
3. Если хвост не найден — добавляет красную надпись "No tail detected"

**Принимает:** изображение (массив пикселей от Gradio), порог уверенности.
**Возвращает:** изображение с визуализацией.

### `segment_video(video_path, confidence=0.3) -> str`

Обработка видео покадрово. Вызывается при нажатии кнопки "Segment" во вкладке Video.

1. Открывает видео через `cv2.VideoCapture`, считывает FPS и разрешение
2. Создаёт временный файл для записи результата (кодек mp4v)
3. В цикле: читает кадр → прогоняет модель → рисует маски → записывает в выходное видео
4. Освобождает ресурсы (`release`)

**Принимает:** путь к видеофайлу, порог уверенности.
**Возвращает:** путь к обработанному видео.

## Интерфейс

Построен на `gr.Blocks` с двумя вкладками:

- **Image** — загрузка фото (`gr.Image`), кнопка "Segment", результат (`gr.Image`)
- **Video** — загрузка видео (`gr.Video`), кнопка "Segment", результат (`gr.Video`)

Слайдер `Confidence threshold` (0.1–0.9, шаг 0.05, по умолчанию 0.3) — общий для обеих вкладок.

Связка интерфейса с функциями через `btn.click(fn, inputs, outputs)`.
