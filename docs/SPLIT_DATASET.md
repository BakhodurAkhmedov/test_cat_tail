# split_dataset.py

Разбиение размеченного датасета на тренировочную (80%) и валидационную (20%) выборки. Модель обучается на train, а качество проверяется на val — данных, которые модель не видела при обучении.

## Константы

- `SPLIT_RATIO = 0.8` — доля тренировочной выборки
- `SEED = 42` — фиксированный seed для воспроизводимости разбиения
- `IMAGE_EXTS = {".jpg", ".jpeg", ".png"}` — допустимые расширения изображений

## Функции

### `main()`

1. **Сбор пар** — проходит по `data/raw/`, для каждого изображения ищет одноимённый `.txt` файл с разметкой. Изображения без разметки игнорируются
2. **Перемешивание** — `random.shuffle` с фиксированным seed. Без перемешивания в val могут попасть только фото с определённых букв алфавита
3. **Разбиение** — первые 80% пар → train, остальные 20% → val
4. **Очистка** — удаляет все файлы из целевых папок (`data/images/train/`, `data/images/val/`, `data/labels/train/`, `data/labels/val/`), чтобы не осталось данных от предыдущего разбиения
5. **Копирование** — `shutil.copy2` копирует файлы из `data/raw/` в соответствующие папки. Оригиналы остаются нетронутыми

## Итоговая структура

```
data/
├── raw/                  # оригиналы (не трогаем)
├── images/
│   ├── train/            # 80% картинок
│   └── val/              # 20% картинок
└── labels/
    ├── train/            # разметка для train
    └── val/              # разметка для val
```
