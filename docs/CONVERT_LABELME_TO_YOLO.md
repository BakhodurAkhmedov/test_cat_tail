# convert_labelme_to_yolo.py

Конвертация разметки из формата LabelMe (JSON с пиксельными координатами) в формат YOLO segment (TXT с нормализованными координатами).

## Форматы

**Вход — LabelMe JSON:**
```json
{
  "imageWidth": 800, "imageHeight": 600,
  "shapes": [{"label": "tail", "shape_type": "polygon", "points": [[120.5, 340.2], ...]}]
}
```

**Выход — YOLO TXT:**
```
0 0.150625 0.567000 0.168750 0.593000 ...
```
`class_id` + нормализованные координаты полигона (x/width, y/height).

## Константы

- `RAW_DIR` — путь к `data/raw/`, где лежат оригиналы и JSON-разметка
- `CLASS_MAP = {"tail": 0}` — соответствие имени класса и числового ID

## Функции

### `convert_one(json_path) -> str | None`

Конвертирует один JSON файл в строку YOLO-формата.

1. Парсит JSON, извлекает размеры изображения
2. Для каждого shape проверяет: метка есть в `CLASS_MAP` и тип — `polygon` (остальное пропускается с предупреждением)
3. Нормализует координаты: делит x на ширину, y на высоту
4. Формирует строку: `class_id x1 y1 x2 y2 ...`

**Возвращает:** строку разметки (или несколько строк через `\n`, если на фото несколько хвостов). `None` — если нет валидных полигонов.

### `main()`

1. Находит все `.json` файлы в `data/raw/`
2. Конвертирует каждый через `convert_one`
3. Сохраняет результат в `.txt` файл с тем же именем (`cat_01.json` → `cat_01.txt`)
4. Выводит статистику: сколько файлов конвертировано из общего количества
